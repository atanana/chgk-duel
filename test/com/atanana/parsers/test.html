MP - прогнозируемое место команды на турнире. Прогноз проводится на основании
рейтингов всех участвующих команд, причём, в зависимости от тех или иных условий,
используется либо релизный рейтинг, либо технический рейтинг по базовому либо
фактическому составу команды. Результат турнира для команды D=U(∙D1+D2),
где U – коэффициент влияния турнира (U=1 для очных турниров, U=2/3 для онлайн-турниров; U=0,5 для прочих синхронов);
D1 – балансная разность: D1=k(B-BP),
где k – коэффициент учёта отрицательных разностей (при BP<B k=1; при BP>B k=0,5 до 1 сентября 2006 и k=0,6 после 1 сентября 2006);
B - турнирный балл, рассчитанный по занятому командой месту;
BP - прогнозируемый турнирный балл команды;
D2 – добавочный бонус D2=F∙exp((B-Bmax)/W),
где F=~300 – максимальный добавочный бонус (может быть разным для разных турниров), Bmax = 2300 - максимальный турнирный балл, W=350 - знаменатель экспоненты.

[123] - Не учитывающийся в релизах балл. PRAT (Presumed Rating After Tournament) - Предполагаемый
рейтинг команды после учёта результатов турнира (может не совпадать
с релизным из-за возможного пересчёта релизного рейтинга с другими опциями) Город, в котором проводился турнир.
(Город) - площадка, где команда сыграла синхрон. Приведено количество взятых командой на турнире вопросов / Количество вопросов на турнире.
Количество показано только для тех турниров, для которых в системе существуют расплюсовки. RG - рейтинг команды на турнире. Равен:

1. до 01.09.2006 - релизному рейтингу команды RG=R;
2. после 1.09.2006 - техническому рейтингу её фактического состава RT в следующих случаях:
- если команда сборная или беспреемственная;
- если релизный рейтинг R=0 и/или технический рейтинг по базовому составу RB=0.
3. для остальных команд - R∙RT/RB (но не менее чем 0.5∙R, если RT<RB, и не более, чем RT в остальных случаях)
